### 🌟 Project Overview
* **ENG:** Built with **PhoBERT (VinAI)** • Fine-tuned for **Vietnamese School Bullying Detection**. *(For research and school violence prevention purposes only.)*
* **VIE:** Được xây dựng bằng **PhoBERT (VinAI)** • Fine-tuned phát hiện **Bạo lực học đường** tiếng Việt. *(Dành cho mục đích nghiên cứu và phòng chống bạo lực học đường.)*

 ENG: **How to install**
 * **1. Clone the repository**: `git clone https://github.com/Dang2nek/FANG-model-for-PhoBERT.git`
  >**In case download fails due to LFS bandwidth limits, please download via Google Drive: [Link downloads model](https://drive.google.com/file/d/1aOfdCv2JkxZ4osw6Q69CsCH4cGKY2YTd/view?usp=sharing)**

 * **3. Start the backend server**: `python server.py`
 * **4. Launch the UI**: `Open blhd_warning.html`
